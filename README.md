# Android ExpensetrackerAPP
## 项目简介
本项目是一个基于 Java 的 Android 记账 App，用于记录日常收入和支出。

## 主要功能
1. 添加消费记录
2. 显示消费记录的具体时间，类型，以及备注
3. 使用本地存储或数据库保存数据

## 使用技术
1. Java
2. Android Studio
3. XML Layout
4. RecyclerView
5. Git

## 项目结构
MainActivity.java: 主页面与用户交互
Expense.java: 账单数据实体类
AppDatabase.java: 数据库入口
ExpenseAdapter.java: 账单列表适配器
